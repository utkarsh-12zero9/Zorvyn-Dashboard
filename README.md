# Setup Instructions
To get the Zorvyn Dashboard running locally, follow these steps:

## Prerequisites
- Node.js 18.x or higher
- npm or yarn

## Installation

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open the browser**:
   Navigate to [http://localhost:3000](http://localhost:3000) to view the dashboard.
